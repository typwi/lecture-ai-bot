# 🎓 AI Lecture Assistant Bot

**Интеллектуальный Telegram-бот для автоматизации работы с лекциями.** Бот мгновенно превращает голосовые заметки в структурированный текст (Whisper) и позволяет общаться с материалом лекции в режиме реального времени (Llama 3.3).

---

## ✨ Ключевые возможности

* 🎙 **Мгновенная транскрипция**: Отправьте аудиосообщение, и Whisper-large-v3 переведет его в текст с учетом пунктуации.
* 🧠 **Контекстное обучение**: Бот хранит историю последних 20 сообщений, позволяя задавать уточняющие вопросы по материалу.
* 📝 **Режим конспекта**: Использование Llama-3.3-70b для суммаризации и выделения главных тезисов из лекции.
* 🧹 **Управление сессиями**: Команда `/new` для обнуления памяти и начала новой темы.
* 📑 **Полный лог**: Команда `/ras` для получения полной расшифровки последнего блока аудио.

---

## 🔑 Подробная настройка API

Для работы бота вам понадобятся два ключа. Вот как их получить:

### 1. Telegram Bot Token

1. Напишите **[@BotFather](https://t.me/BotFather)** в Telegram.
2. Введите команду `/newbot`, выберите имя и уникальный юзернейм для бота.
3. Сохраните полученный **HTTP API Access Token**.

### 2. Groq Cloud API (Бесплатно)

1. Перейдите в **[Groq Cloud Console](https://console.groq.com/keys)**.
2. Авторизуйтесь и перейдите в раздел **API Keys**.
3. Нажмите **Create API Key**, назовите его (например, `LectureBot`) и скопируйте.
> *Примечание: Groq обеспечивает невероятную скорость генерации (до 300+ токенов в секунду), что делает общение с ботом бесшовным.*

## 🚀 Быстрый старт

### Шаг 1: Клонирование репозитория

```bash
git clone https://github.com/ваш-аккаунт/ai-lecture-bot.git
cd ai-lecture-bot

```

### Шаг 2: Настройка окружения

Рекомендуется использовать виртуальное окружение, чтобы избежать конфликтов библиотек.

```bash
python -m venv venv

# Для Windows:
venv\Scripts\activate

# Для macOS/Linux:
source venv/bin/activate

pip install -r requirements.txt

```

### Шаг 3: Конфигурация

Создайте файл `config.py` на основе примера:

```bash
cp config.py.example config.py

```

Откройте его и вставьте свои данные:

```python
# config.py
TELEGRAM_TOKEN = "123456789:ABCDefGhIJKlmNoP..."
GROQ_API_KEY = "gsk_YourActualGroqKeyHere..."

```

### Шаг 4: Запуск

```bash
python main.py

```

## 💡 Использование

1. **Запишите лекцию** на диктофон или отправьте голосовое сообщение боту.
2. Дождитесь уведомления «Текст распознан».
3. Задайте вопрос: *"Какие три главных вывода были в этой части?"* или *"Составь план по этому фрагменту"*.
4. Используйте `/new`, когда переходите к следующей паре или теме.
